# css-outline-debug
JavaScript-скрипт для визуализации границ HTML-контейнеров на сайте. Удобен для проверки вёрстки, структуры блоков и отладки layout прямо в браузере.
# Outline Containers Debug Script

A small JavaScript utility to visualize HTML container boundaries directly in the browser.  
Useful for designers, frontend developers, and layout debugging.

---

## Features

- Shows outlines for all HTML elements
- Toggle ON / OFF without page reload
- Works on any website
- No dependencies
- Runs directly from DevTools or as a bookmarklet

---

## Usage (DevTools)

1. Open DevTools (F12)
2. Go to the Console tab
3. Paste and run the script
